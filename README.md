### Server
Para iniciar o servidor utilize o seguinte comando: `make server` *OU* `python3 Server/main.py`

### Jogadores
Para criar uma partida no servidor ou conectar em uma, utilize o comando: `make client` *OU* `python3 Client/main.py`
